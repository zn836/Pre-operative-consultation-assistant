# 快速入门指南 ⚡

5分钟快速上手术前谈话助手!

## 快速安装 (3步骤)

### 步骤 1️⃣: 安装依赖

```bash
# 进入项目目录
cd video_qa_assistant

# 安装 Python 包
pip install streamlit requests python-dotenv
```

### 步骤 2️⃣: 准备视频

将视频和字幕文件放入 `video` 文件夹:
```
video/
├── 1.mp4      # 视频文件
└── 1.txt      # 字幕文件
```

字幕格式示例:
```
[00:00:00.620 - 00:00:12.300] 各位患者和家属大家好
[00:00:12.300 - 00:00:25.500] 今天讲术前准备
```

### 步骤 3️⃣: 启动应用

```bash
streamlit run app.py
```

浏览器自动打开 `http://localhost:8501` ✅

---

## 快速使用 (3步骤)

### 1. 提出问题
在输入框输入: **"术前需要做哪些准备?"**

### 2. 查看回答
- 系统会定位到相关视频时间点
- 显示内容概要

### 3. 深入了解
- 点击 **"📍播放"** → 跳转视频
- 点击 **"👨‍⚕️ 获取专家详细解答"** → 查看详细指导

---

## 常用问题示例

✅ "术前需要做哪些检查?"
✅ "抗凝药物需要停用吗?"

## 推荐使用流程

```
第1步: 完整观看视频 → 了解整体内容
第2步: 查看推荐问题 → 点击感兴趣的问题
第3步: 针对性提问 → 深入了解细节
第4步: 视频跳转观看 → 反复学习重点内容
第5步: 获取专家解答 → 详细理解指导
```

---

## 常见问题快速解决

| 问题 | 解决方法 |
|------|---------|
| 命令不存在 | `python -m streamlit run app.py` |
| 视频不播放 | 检查格式是否为 mp4/avi/mov |
| 找不到答案 | 换一种方式提问,或查看推荐问题 |
| 响应很慢 | 正常需要2-10秒,检查网络连接 |
